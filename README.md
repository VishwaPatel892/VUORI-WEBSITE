# 🏃‍♂️ VUORI WEBSITE

This project is a multi-page e-commerce website inspired by Vuori, featuring product-focused pages such as Activewear, Men, Women, and Collections. The website is designed with a modern, minimal UI, smooth navigation, and interactive elements like hover effects to enhance user experience.
